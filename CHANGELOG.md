## [1.0.0] - 2026-04-27

### Added
- **MCP Server** — Full Model Context Protocol server with 24 tools for email management
- **VS Code Extension** — Companion extension with embedded webview for auth, review, and settings
- **React Webview** — Auth flow, inbox review panels, settings, and built-in documentation
- **Human-in-the-Loop Review System** — AI proposes actions, you approve or reject in a visual checklist
- **Email Operations** — List, search, read, move, and delete emails via MCP tools
- **Folder Management** — List and tree views of mail folder hierarchy
- **High-Performance Cache** — Sync hundreds of emails to memory for instant search and filtering
- **Bulk Unsubscribe** — Detect and execute unsubscribe for mailing lists
- **Device Code Auth Flow** — Secure OAuth 2.0 sign-in via browser with auto-copy and auto-open
- **Microsoft Exchange Provider** — Full support for Outlook / Microsoft 365 mailboxes
- **Provider-Agnostic Architecture** — Plugin system for adding new email providers
- **Custom AI Instructions** — Override or extend the AI's default behavior from settings
- **MCP Server Auto-Management** — Start, stop, restart the server from the extension panel
- **Parallel Tool Execution** — MCP server supports concurrent tool calls and multiple reviews
- **Built-in Documentation** — Overview, getting started, capabilities, tools, workflows, and connection guides
